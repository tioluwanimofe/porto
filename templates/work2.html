002 : Autonomous robot

This webpage showcases the winning robot designed for the Trevecca Nazarene University Engineering Design Class Project Competition. The robot excels at placing blocks into designated areas and is controlled wirelessly.

Robot Specifications


Functionality:


- Places blocks into designated areas
- Controlled wirelessly via remote control
- Locomotes for movement

Components:


Robot Shield:
- Arduino Uno R3
- Servos (2)
- Multicolored LEDs
- Resistors, Transistors
- DC Motors (2)
- H-bridge
- Female Jumper Wires
- RF24 wireless communicator (receiver)
- 5V Battery Pack
- Controller Shield:
Arduino Uno R3
LEDs Resistors, Transistors
Multicolored push buttons (6)
Joystick 5V Battery Pack
Female Jumper Wires
RF24 wireless communicator (transmitter)
Robotic Structure:
Designed using SOLIDWORKS software
3D printed parts
Assembled with soldering and minor CNC modifications

Design Process


Ideation phase: Brainstorming for two weeks to develop the robot design
Design and Prototyping phase:
Building and testing the robot shield and controller shield
Programming the Arduino Uno R3 for movement and speed control
This winning robot demonstrates the successful application of design principles and engineering skills.
